# WXV - WebX Viewer

View and use WebX websites comfortably from your browser using WXV.

**Features:**

- Tabs
- 4 Way tab bar placement
- Browser themes
- Website themes
- Legacy lua support
- Bussinga css/lua emulation
- History
- And more!
